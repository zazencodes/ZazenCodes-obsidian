# Simple minimalist logo prompt

I used this prompt to make a minimalist samurai-crest inspired logo that turned out pretty good

https://chatgpt.com/c/6937e4d1-29e8-832d-ad6c-c290eca9cba7

The prompt:

```
Riff on this and create a minimalist black-and-white logo on transparent background. Square layout. Some 3D elements to create a professional feel.
```

And I uploaded a reference image of the samurai family crest I was interested
in. Something from [this site](https://samuraistore.com/pages/crests?srsltid=AfmBOop4rb742YsOzTAuwVDOGWWc8j_qYWmla_bLu3YlfXx7_PyLoF8o)
